

## 概要

[Flutter Tokyo#11](https://flutter-jp.connpass.com/event/374220/)の登壇資料
[flutter_deck](https://pub.dev/packages/flutter_deck)を使用して作成しています。

## 実行方法

```bash
flutter run --dart-define-from-file .env
```
※ .env.exampleをコピーして.envを作成し、APIキーを設定してください。

## スライド構成

1. タイトル
2. 自己紹介
3. flutter_genuiとは？
4. Geminiによる生成UIの活用
5. flutter_genuiの仕組み
6. CatalogItem定義
7. Conversation作成
8. Surface表示
9. デモ
10. 参考文献
11. 終わり
